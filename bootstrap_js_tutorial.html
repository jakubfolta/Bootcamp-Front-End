Getting started with Bootstrap!

Bootstrap is the most popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web.

Tooltips

Inspired by the excellent jQuery.tipsy plugin written by Jason Frame, tooltips are an updated version, which don't rely on images - they use CSS3 for animations and data-attributes for local title storage.

Markup

The required markup for a tooltip is only a data attribute and a title on the HTML element you wish to have a tooltip on. The generated markup of a tooltip is rather simple, though it does require a position setting (by default, set to top by the plugin).
